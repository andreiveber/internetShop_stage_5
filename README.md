# Интернет-магазин (учебный проект) - Этап 5

## Описание проекта

На данном этапе в проект добавлены дополнительные варианты маршрутов, создана область (Area) для административной панели, реализованы атрибуты маршрутов и передаваемые параметры через URL.

## Функциональность

В проекте реализовано:
- контроллер `HomeController` с тремя действиями (actions);
- модель `Product` для передачи данных;
- макет `_Layout.cshtml` для всех страниц с навигацией;
- область Admin с отдельным контроллером и представлениями;
- пять страниц (представления):
  - **главная страница (Index)** — приветствие и описание магазина;
  - **страница с товарами (Products)** — динамический список товаров из модели с возможностью поиска по названию;
  - **страница контактов (Contact)** — адрес, телефон, email, часы работы;
  - **страница "О нас" (About)** — информация о компании;
  - **панель администратора (Admin/Dashboard)** — статистика и управление.

## Новые маршруты
- добавлен псевдоним `/catalog` для страницы товаров;
- добавлен маршрут `/product/{id}` для детального просмотра товара;
- добавлен маршрут `/feedback/{name?}` с опциональным параметром;
- добавлен маршрут `/about для страницы` "О компании";
- создана область Admin с маршрутами `/admin/dashboard`, `/admin/dashboard/stats`, `/admin/dashboard/product/{id}`.

##  Атрибуты маршрутов в контроллерах
- `[Route("catalog")]` — псевдоним для страницы товаров;
- `[Route("product/{id:int}")]` — детальная страница с параметром; 
- `[Route("feedback/{name?}")]` — опциональный параметр в URL;
- `[Route("about")]` — страница "О компании";
- `[Area("Admin")]` и `[Route("admin/[controller]")]` — настройка области администратора.
