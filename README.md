## Описание проекта

На данном этапе в проект добавлены новые интерактивные компоненты Bootstrap, улучшена навигация, кастомизированы цвета и реализована полноценная адаптивность под все устройства.

## Функциональность

В проекте реализовано:
- контроллер `HomeController` с тремя действиями (actions);
- модель `Product` для передачи данных;
- макет `_Layout.cshtml` для всех страниц с навигацией.
- три страницы (представления):
  - **главная страница (Index)** — приветствие и описание магазина;
  - **страница с товарами (Products)** — динамический список товаров из модели с возможностью поиска по названию;
  - **страница контактов (Contact)** — адрес, телефон, email, часы работы.

## Новые компоненты Bootstrap
- добавлен пункт меню "Категории" с выпадающим списком;
- пункты меню: "Игры для PS5", "Игры для Xbox", "Игровые консоли", "Аксессуары";
- кастомизированные цвета при наведении.

## Карусель на главной странице
- автоматическая прокрутка изображений;
- три слайда;
- кастомные синие стрелки управления;
- синие индикаторы с эффектом увеличения при наведении;
- фиксированное соотношение сторон для сохранения пропорций.

## Детали реализации

### Макет (_Layout.cshtml)
- подключение Bootstrap 5.3 через CDN;
- адаптивная навигационная панель с логотипом SOTCOMM;
- единая шапка с ссылками на все страницы (Главная, Категории, Товары, Контакты);
- подвал с информацией об авторских правах;
- динамический заголовок через `ViewData["Title"]`

### Страница товаров
- данные передаются из контроллера; 
- форма поиска с фильтрацией по названию;
- таблица со стилизацией Bootstrap;
- чередование фона строк и подсветка при наведении;
- счетчик общего количества и найденных товаров;
- ссылка для сброса поиска.

### Страница контактов
- адрес магазина;
- телефон;
- email;
- часы работы.
