# Интернет-магазин (учебный проект) - Этап 3

## Описание проекта

На данном этапе в проект интегрирован Bootstrap 5.3, реализована адаптивная сетка и улучшен пользовательский интерфейс.

## Функциональность

В проекте реализовано:
- контроллер `HomeController` с тремя действиями (actions);
- модель `Product` для передачи данных;
- макет `_Layout.cshtml` для всех страниц с навигацией.
- три страницы (представления):
  - **главная страница (Index)** — приветствие и описание магазина;
  - **страница с товарами (Products)** — динамический список товаров из модели с возможностью поиска по названию;
  - **страница контактов (Contact)** — адрес, телефон, email, часы работы.

## Детали реализации

### Макет (_Layout.cshtml)
- подключение Bootstrap 5.3 через CDN;
- адаптивная навигационная панель с логотипом;
- единая шапка с ссылками на все страницы (Главная, Товары, Контакты);
- подвал с информацией об авторских правах;
- динамический заголовок через `ViewData["Title"]`

### Страница товаров
- данные передаются из контроллера; 
- форма поиска с фильтрацией по названию;
- таблица со стилизацией Bootstrap;
- чередование фона строк и подсветка при наведении;
- счетчик общего количества и найденных товаров;
- ссылка для сброса поиска.

### Страница контактов
- адрес магазина;
- телефон;
- email;
- часы работы.
